# ToDoList App

**React**와 **styled-components**를 사용하여 만든 간단한 ToDoList 애플리케이션으로, 사용자가 작업을 가져오고, 추가하고, 완료하고, 삭제할 수 있습니다.

## 프로젝트 실행 방법

### 설치 종속성
- **npm:**
  ```bash
  npm install
- **yarn:**
  ```bash
  yarn install

### 개발 서버 시작

- **npm:**
  ```bash
  npm start
- **yarn:**
  ```bash
  yarn start
